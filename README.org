*NASCTY.jl* is a Julia library for breaking cryptographic targets using Side-Channel Analysis (SCA)
It implements a neural architecture search (NAS) strategy based on a genetic algorithm, tailored for side-channel analysis.
It is based upon the research: /Neuroevolution to Attack Side-Channel Traces Yielding convolutional neural networks/ (NASCTY) [1].

Project goals:
- Easily adapt and configure the search to your side-channel analysis dataset;
- Be a simple to understand implementation of neural architecture search for the SCA domain.
- Give clear actionable insights into the search results.

* Installation
Requirements:
- Julia >= 1.12

Install via the Julia REPL (press ~]~ to enter Pkg mode).
Installing the stable version can be done with:
#+BEGIN_SRC julia :eval noeval
pkg> add NASCTY
#+END_SRC

As our latest commit this is not yet an option, if you wish access to the library already, an alternative route to the latest development version can be gained by:
#+BEGIN_SRC julia :eval noeval
pkg> add https://github.com/Peter-van-der-Velde/NASCTY.jl
#+END_SRC

* Usage
To start a neural architecture search on your side-channel analysis dataset, load your data and initialise the hyperparameters:
#+BEGIN_SRC julia :eval noeval
import NASCTY
import Flux

@info "Setting NASCTY hyperparameters" hyperparameters =
NASCTY.NASCTYGENOMEHYPERPARAMETERS()

@info "Starting neural architecture search..." generations =
NASCTY.nascty(hyperparameters, trainingdataloader, validationdata)

@info "Finished neural architecture search..."
bestgenomes = generations .|> generation -> argmin(g -> g.loss, generation)
bestgenome = argmin(g -> g.loss, bestgenomes)

@info "The best genome found is:" bestgenome
#+END_SRC

For detailed walkthroughs, please visit the [[file:examples/][examples directory]]
and its accompanying [[file:examples/README.org][README.org]] for more information.
